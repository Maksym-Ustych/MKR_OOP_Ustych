# MKR_OOP_Ustych

Модульна контрольна робота з дисципліни  
«Об’єктно-орієнтоване програмування».

## Автор
Устич Максим  
Група: alk-43

## Опис роботи
Репозиторій містить виконані завдання модульної контрольної роботи з ООП.

Робота складається з трьох рівнів складності:

- Рівень 1 — теоретичні питання та приклади програмної реалізації
- Рівень 2 — патерни проєктування та принципи ООП
- Рівень 3 — практичні задачі мовою C++

## Використані технології
- C++
- Visual Studio 2022
- Об’єктно-орієнтоване програмування
- Шаблони
- Віртуальні функції
- Поліморфізм
- Патерни проєктування

## Структура репозиторію
- MKR Level 1
- MKR Level 2
- MKR Level 3
- PDF-звіт роботи
- Вихідні коди програм
